# Portfólio Pessoal | Leonardo Lima

Este é um projeto de portfólio pessoal desenvolvido para apresentar informações sobre formação, projetos e permitir o contato direto. O site foi construído utilizando tecnologias web fundamentais, focando em uma experiência de usuário limpa, responsiva e moderna.

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica das páginas.
- **CSS3**: Estilização moderna com variáveis (Custom Properties), Flexbox, Grid e animações.
- **JavaScript (Vanilla)**: Lógica para interatividade, manipulação de DOM e persistência de dados local.

## Funcionalidades

- **Menu Responsivo**: Navegação adaptada para dispositivos móveis com menu hambúrguer interativo.
- **Modo Escuro/Claro**: Alternância de tema com persistência automática via `localStorage`.
- **Validação de Formulário**: Sistema de validação de campos no formulário de contato para garantir a integridade dos dados enviados.
- **Design Moderno**: Uso da tipografia 'Inter' e transições suaves para uma experiência premium.

## Estrutura do Projeto

```text
.
├── index.html          # Página inicial (Hero)
├── sobre.html          # Informações biográficas
├── formacao.html       # Detalhes acadêmicos e cursos
├── portfolio.html      # Galeria de projetos
├── contato.html        # Formulário de contato
├── css/
│   └── style.css       # Estilos globais e responsivos
└── js/
    └── script.js       # Lógica de interatividade e tema
```

## Como Executar o Projeto

1. Clone o repositório (após o upload):
   ```bash
   git clone https://github.com/leogomeslima/portifolio-pessoal.git
   ```
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
